    - 移行元 DB からダンプファイルを取得し、Blob→Hex 変換後に移行先 DB へ取り込み、UPDATE 文でデータを復旧しました
    - Collation が誤って変更されていることを特定し、`utf8mb4_unicode_ci` に修正して解決しました
    - すべての障害対応をその日中に完了し、ユーザー影響を最小限に抑えました
